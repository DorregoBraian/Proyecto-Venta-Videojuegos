Bienvenido a GameStore

Te presentamos nuestro proyecto de portafolio: una tienda de videojuegos construida con tecnología de última generación para brindar una experiencia visual y funcional de alta calidad. Desde la creación de un backend propio en .NET hasta la implementación de patrones avanzados en Angular 18, este sitio representa la culminación de técnicas de desarrollo cuidadosamente seleccionadas y optimizadas para la velocidad y la experiencia de usuario.
